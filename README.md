cat > README.md << 'EOF'
# 🎬 Movie Recommendation System

A content-based Movie Recommendation System built using Python, NLP and Machine Learning.

## Features
- Recommends 5 similar movies based on your selection
- Fetches movie posters using TMDB API
- Built with Streamlit for interactive UI

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK (Stemming)
- CountVectorizer
- Cosine Similarity
- Streamlit
- TMDB API

## How It Works
1. Select a movie from the dropdown
2. Click Recommend button
3. Get 5 similar movie recommendations with posters

## Dataset
TMDB 5000 Movies Dataset

## Run Locally
streamlit run app.py
EOF
